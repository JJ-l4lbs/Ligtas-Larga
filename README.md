# Ligtas-Larga

Ligtas-Larga (Ligtas-Lakbay) is a mobile-first crowdsourced mapping and navigation web application. It is designed to assist urban commuters, students, and vulnerable sectors in identifying and reporting accessibility hazards and environment issues (such as flooding) in real time.

## Key Features
- **Mobile-First Architecture:** The client interface, layout drawers, maps canvas, and user flows are built specifically for mobile screens and touch environments first.
- **Multi-Profile Routing:** Tailored routing modes for wheelchair/stroller accessibility, students (shaded paths), and weather/rain alerts.
- **Smart Hazard Verification:** Crowdsourced hazard submissions validated using the Hugging Face Inference API before pin markers are posted to the map.
